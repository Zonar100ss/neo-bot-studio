Политика конфиденциальности

1. Сбор и использование данных

Приложение не собирает, не хранит и не передаёт ваши персональные данные. Все операции выполняются локально на вашем устройстве в браузере.

2. Локальное хранение информации

Все файлы, боты, чаты и настройки сохраняются исключительно в localStorage вашего браузера. Данные не синхронизируются с серверами и не передаются третьим лицам.

Важно: При очистке данных браузера вся информация приложения будет удалена. Рекомендуем регулярно экспортировать резервные копии.

3. Использование внешних API

При использовании онлайн-режима (не локальной модели) запросы к ИИ-провайдерам (Hugging Face, OpenAI и др.) отправляются напрямую с вашего устройства, минуя серверы разработчика.

Вы можете использовать собственные API-ключи для полного контроля над запросами.

4. Открытость и прозрачность

Neo-Bot Studio — это полностью бесплатное приложение с открытым исходным кодом. Исходный код доступен для аудита в любое время:

  • 📄 GitHub репозиторий
  • 📚 Документация
  • 🐞 Сообщить об ошибке

О разработчике

Neo-Bot Studio разработан Zonar100 — энтузиастом приватных ИИ-решений. Проект создан с целью предоставить пользователям мощный инструмент для работы с ИИ без компромиссов в области конфиденциальности.

Локальная модель ИИ Qwen

Полная приватность

Локальная модель работает полностью офлайн на вашем устройстве. Ваши данные никогда не покидают браузер.

Требуется загрузка

Модель Qwen2.5-0.5B (~512 МБ) будет загружена один раз и сохранена в кэше браузера.

⚠️ Требуется стабильное интернет-соединение для первой загрузки (~2-5 минут)

Преимущества

  • ✅ Все диалоги полностью приватны и офлайн
  • ✅ Нет ограничений на количество запросов
  • ✅ Работает даже без интернета (после загрузки)
  • ✅ Быстрый отклик без задержек серверов
  • ✅ Подходит для работы с конфиденциальной информацией

Вопросы и ответы

Будет ли модель работать без интернета?
Да! После первой загрузки все диалоги работают полностью офлайн.

Сколько места займёт модель?
~512 МБ в кэше браузера (можно очистить в настройках браузера).

Могу ли я отключить модель позже?
Да, в настройках можно в любой момент переключиться на онлайн-режим.
