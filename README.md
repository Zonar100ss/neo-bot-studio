# neo-bot-studio
Приложение-студия для ролевых игр (RP) с поддержкой модов и плагинов. Интерактивная платформа для ролевиков, предоставляющая гибкие инструменты для кастомизации игрового процесса. Отличительная черта — открытая архитектура: пользователи могут подключать плагины, устанавливать моды и интегрировать собственные Ai модели через специальное API.
