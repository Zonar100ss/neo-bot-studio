## 🌟 О приложении

**Neo-Bot Studio** — это инновационное веб-приложение премиум-класса для создания, тонкой настройки и управления персональными ИИ-ассистентами нового поколения. Приложение работает **исключительно в среде вашего браузера**, не требует подключения к внешним серверам и гарантирует, что ваши диалоги, конфигурации ботов и персональные данные **ни при каких обстоятельствах не покидают ваше устройство**.

> 🔐 *«Ваша приватность — не функция, а фундаментальная архитектура»*

- 🎨 Дизайн: Glassmorphism + Neon Glow
- 🧠 ИИ-ядро: Локальная модель Qwen 2.5
- 🔌 Расширяемость: Система плагинов JS
- 💾 Хранение: localStorage браузера
- 🌐 Доступ: Полностью офлайн-режим
- 📱 Формат: PWA с установкой на устройство 

## ✨ Ключевые возможности

🔒 **Приватность архитектурного уровня**
• Все данные хранятся исключительно в `localStorage` вашего браузера.
• Полное отсутствие фоновых запросов и телеметрии.
• Шифрование сессии при экспорте/импорте на стороне клиента.
• 100% исходный код доступен для независимого аудита.

🤖 **Локальное ИИ-ядро на вашем устройстве**
• Поддержка моделей: **Qwen2.5-0.5B-Instruct** (~512 МБ), **Qwen2.5-1.5B-Instruct** (~1.2 ГБ).
• Альтернативы: Phi-3-mini, Gemma-2B, TinyLlama.
• Диалоги остаются на устройстве — **100% конфиденциальность**.
• Полная функциональность без интернет-соединения.
• Мгновенная генерация ответов без сетевых задержек.
• Отсутствие лимитов на количество запросов.

🧩 **Профессиональная система плагинов**
• Категории: Инвентарь, Крафт, Магия, Бой, Квесты, Диалоги, События, Таймеры, Хранение, Утилиты.
• Функции: `createInventory()`, `addItem()`, `craft()`, `castSpell()`, `attack()`, `createQuest()`, `dialogOption()`, `onEvent()`, `setTimer()`, `saveData()`, `random()` и многие другие.
• Плагины создаются внутри приложения и **не считаются производными работами**.

🎨 **Безграничная визуальная кастомизация**
• Темы: Тёмная (Neon Glow) / Светлая (Glass Morphism).
• Персонализация: фон, текст бота, текст пользователя, аватары, фоны чата.
• Анимационная система: настраиваемые переходы, эффекты появления, микро-интеракции.
• Параметры ИИ: температура **0.0–2.0**, длина ответа **256–2048** токенов, контекст **5/10/20** сообщений.
• TTS с поддержкой русского языка и голосовой ввод через Web Speech API.

📱 **Профессиональный пользовательский интерфейс**
• **Glassmorphism UI** с эффектом матового стекла и размытия фона.
• Neon-акценты с плавными анимациями свечения для тёмной темы.
• Адаптивная вёрстка для мобильных, планшетов и десктопов.
• Поддержка `prefers-reduced-motion`, `prefers-contrast`, ARIA-атрибутов.
• Мультиязычность: русский и английский интерфейсы.
• PWA-возможности: установка как нативное приложение с иконкой.

## 🚀 Мгновенный старт

**Вариант 1: Локальный запуск (рекомендуется)**

1. Получите файлы приложения (скачайте архив или клонируйте репозиторий при наличии доступа).
2. Откройте index.html в современном браузере:
   • Chrome 115+ | Firefox 120+ | Edge 115+ | Safari 16.4+
3. Приложение запустится автоматически:
   • Пройдите экран политики конфиденциальности.
   • При желании загрузите локальную модель ИИ.
   • Создайте первого бота и начните диалог.

**Вариант 2: Прямое использование файлов**

📁 Структура дистрибутива:
├── index.html              # Точка входа приложения
├── scripts/
│   └── core/app.js         # Ядро логики и управление состоянием
├── styles/
│   ├── neo-core.css        # Базовые стили и сброс
│   ├── neo-components.css  # Компоненты пользовательского интерфейса
│   ├── neo-animations.css  # Система анимаций и переходов
│   └── themes/
│       ├── dark-neon.css   # Тёмная тема с неоновыми эффектами
│       └── light-glass.css # Светлая тема с glassmorphism
├── assets/                 # Иконки, изображения, шрифты
└── sw.js                   # Service Worker для PWA-функционала

**📲 Установка как прогрессивное веб-приложение (PWA)**

1. Скачайте APK и распакуйте.
3. Приложение появится на рабочем столе с собственной иконкой.
4. После первой загрузки работает полностью офлайн.

## 🛠️ Технические характеристики

**Зависимости (через CDN — без локальной установки)**
• Шрифты: **Inter** для интерфейса, **JetBrains Mono** для кода (Google Fonts).
• Иконки: **Font Awesome 6.4.2** (cdnjs).
• Всё остальное — чистый ванильный JavaScript + CSS.
• Никаких фреймворков, сборщиков или компиляций.

**Структура локального хранения данных**

localStorage keys (префикс: neo_bot_):

• bot_files          — Список идентификаторов созданных ботов

• bot_{id}           — Полные данные конкретного бота

• chat_{id}          — История диалога с ботом

• plugin_{id}        — Код и метаданные плагина

• plugins_list       — Список установленных плагинов

• images_library     — Галерея изображений для ботов

• chat_settings      — Глобальные настройки интерфейса чата

• api_stats          — Статистика сетевых запросов

• privacy_accepted   — Флаг принятия политики конфиденциальности

• qwen_accepted      — Флаг согласия на загрузку локальной модели

**Производительность и оптимизация**
• **Ленивая загрузка** — модули инициализируются только при необходимости.
• **Эффективное кэширование** — повторное использование данных без дублирования.
• **Автоматическая очистка** — управление квотой localStorage при переполнении.
• **Асинхронная обработка** — интерфейс остаётся отзывчивым при генерации ответов.
• **Адаптивная детализация** — автоматическая подстройка под возможности устройства.

## ❓ Часто задаваемые вопросы

**🤔 Работает ли приложение без интернета?**  
Да. После первой загрузки и (опционально) загрузки локальной модели ИИ, приложение функционирует полностью офлайн. Все диалоги, создание ботов и работа плагинов происходят исключительно на устройстве пользователя.

**🔐 Где хранятся мои данные?**  
Все данные сохраняются в `localStorage` вашего браузера. Это изолированное хранилище, доступное только данному приложению в данном браузере на данном устройстве. Данные не синхронизируются с облаком и не передаются третьим лицам.

**🧠 Что такое локальная модель ИИ?**  
Локальная модель — это компактная версия языковой модели (например, Qwen2.5-0.5B), которая загружается в кэш браузера и выполняет генерацию ответов непосредственно на вашем устройстве. Это обеспечивает максимальную приватность и независимость от интернет-соединения.

**🔌 Можно ли расширять функционал?**  
Да, через встроенную систему плагинов на JavaScript. Вы можете создавать собственные функции, механики и интеграции, которые будут работать в рамках вашего экземпляра приложения. Плагины, созданные внутри приложения, **не считаются производными работами** согласно лицензии.

**📱 Можно ли установить на телефон?**  
Да, приложение поддерживает стандарт **PWA** (Progressive Web App). Откройте сайт в мобильном браузере, нажмите **«Добавить на главный экран»** — и приложение будет доступно как отдельная иконка с возможностью работы офлайн.
