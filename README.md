## 📋 Описание

Neo-Bot Studio — это мощное веб-приложение для создания, настройки и управления ИИ-ботами с поддержкой локальной модели Qwen. Полностью работает офлайн, обеспечивает максимальную приватность и не требует серверной части.

## ✨ Особенности

- 🔒 **Полная приватность**: Все данные хранятся только в вашем браузере
- 🤖 **Локальная модель Qwen**: Работа без интернета после первой загрузки
- 🧩 **Система плагинов**: Расширяемый функционал через JavaScript
- 🎨 **Гибкая кастомизация**: Темы, цвета, аватары, фоны
- 📱 **PWA поддержка**: Установка как нативное приложение
- 🌐 **Мультиязычность**: Поддержка русского и английского языков
- 💾 **Экспорт/Импорт**: Резервное копирование всех данных
